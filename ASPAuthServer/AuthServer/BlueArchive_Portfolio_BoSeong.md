### 4. ASP.NET Core 인증 서버 (게임 서버 개인 프로젝트)

| 항목 | 내용 |
|------|------|
| 개발 기간 | 2025년 12월 15일 ~ 2025년 12월 29일 |
| 역할 | 개인 프로젝트 |
| 주요 기술 | ASP.NET Core 8.0, C#, MySQL, Redis, JWT |

#### 프로젝트 개요
게임 서버 인증을 위한 RESTful API 기반 인증 서버입니다. JWT 토큰 기반 다단계 인증 체계를 구현하여 보안성과 확장성을 확보했으며, MySQL과 Redis를 활용한 효율적인 데이터 관리를 구현했습니다.

#### 핵심 구현 사항

**인증 시스템**
- **3단계 토큰 체계**: Login Token → Access Token + Refresh Token 방식 구현
  - Login Token: 로그인 시 1회성 토큰 발급 (재사용 방지)
  - Access Token: 짧은 수명의 API 접근 토큰 (stateless)
  - Refresh Token: Access Token 갱신용 장기 토큰 (Redis 저장)
- **계정 보안**: 로그인 실패 횟수 제한 및 계정 잠금 메커니즘
- **비밀번호 해싱**: SHA256 다중 반복 해싱 (설정 가능한 반복 횟수)
- **디바이스 기반 세션 관리**: DeviceId를 활용한 멀티 디바이스 로그인 지원

**데이터베이스 설계**
- **MySQL**: 사용자 계정 정보, 인증 기록 영구 저장
- **Redis**: 토큰 상태 관리 및 세션 데이터 캐싱
  - Login Token: 활성/사용됨 상태 추적 (재사용 공격 방지)
  - Refresh Token: 디바이스별 세션 관리
  - TTL 자동 관리로 만료된 토큰 자동 삭제

**아키텍처 및 설계 패턴**
- **레이어드 아키텍처**: Controller → Service → Repository 계층 분리
- **의존성 주입 (DI)**: ASP.NET Core의 DI 컨테이너 활용
- **설정 파일 분리**: appsettings.json을 통한 환경별 설정 관리
- **팩토리 패턴**: 토큰 타입별 생성 로직 캡슐화 (TokenFactory)

#### 주요 API 엔드포인트
- `POST /api/auth/register`: 회원가입 (중복 검증, 비밀번호 강도 확인)
- `POST /api/auth/login`: 로그인 (Login Token 발급)
- `POST /api/auth/exchange`: Login Token → Access/Refresh Token 교환
- `POST /api/auth/refresh`: Access Token 갱신 (Refresh Token 사용)
- `POST /api/auth/logout`: 로그아웃 (모든 토큰 폐기)

#### 보안 기능
- **토큰 재사용 공격 방지**: Login Token 1회용 처리, 재사용 시도 로깅
- **계정 보호**: 로그인 실패 5회 시 계정 30분 잠금 (설정 가능)
- **JWT 서명 검증**: 대칭키 기반 토큰 무결성 검증
- **TTL 기반 자동 만료**: 토큰별 수명 관리 (Login: 1분, Access: 15분, Refresh: 30일)

#### 기술적 특징
- **.NET 8.0 최신 기능 활용**: Minimal API 스타일 대신 Controller 기반 명확한 구조
- **비동기 프로그래밍**: async/await 패턴으로 I/O 작업 최적화
- **로깅 시스템**: ILogger를 활용한 체계적인 로그 관리
- **타입 안정성**: Nullable Reference Types 활성화로 null 참조 오류 방지

#### 향후 개발 계획
- 이메일 인증 시스템 추가
- OAuth 2.0 소셜 로그인 연동 (Google, Steam)
- Rate Limiting 미들웨어 적용 (DDoS 방어)
- Unit Test 및 Integration Test 작성

게임 기획 데이터를 자동화하기 위한 엑셀 테이블 추출 도구입니다. 기획자가 작성한 엑셀 스프레드시트를 게임에서 사용 가능한 데이터 형식으로 자동 변환하여 개발 프로세스를 간소화했습니다.

#### 주요 기능
- 엑셀 범위 선택 및 테이블 자동 추출
- MSI 설치 패키지 제공으로 팀원 간 쉬운 배포
- Settings.xlsx를 통한 변환 규칙 관리

#### 프로젝트 성과
- 기획 데이터 입력 시간 단축 및 오류 감소
- 이벤트 추가 프로세스 간소화 기여
